# Interviewer Follow-up Generator

Use this skill when the goal is to simulate the next questions an interviewer might ask after the base solution discussion.

## Responsibilities

- Generate realistic follow-up questions tied to the original problem
- Include likely edge cases and common pitfalls
- Push on constraints, scalability, alternative inputs, or implementation details
- Keep the questions relevant to DSA interview prep

## Output Style

- Favor short, natural interviewer-style prompts
- Include variations about constraints, data structure changes, and optimization tradeoffs
- Surface common mistakes the candidate should be ready to address
